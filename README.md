## Installation

* With Deno

```
deno install --allow-read --allow-env --allow-write -n juice --no-check https://denopkg.com/Atticus64/juice@main/src/main.ts
```

* Download binary from [releases page](https://github.com/Atticus64/juice/releases)  

## Dev server

```bash
deno task dev
```

## Build

```
deno task build
```
